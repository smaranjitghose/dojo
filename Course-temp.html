Web Development Bootcamp

Master the fundamentals of web development with HTML, CSS, and JavaScript.

Pre-requisites

  • Basic Programming Concepts (Variables, Loops, Functions)
  • Understanding of File & Folder Structures
  • Basic Terminal/Command Line Usage
  • Familiarity with Version Control (Git & GitHub Basics)
  • Basic Understanding of HTTP and How the Web Works
  • Text Editor or IDE Experience (VS Code, Sublime, etc.)

Learning Objectives

  • Build responsive websites
  • Understand frontend and backend development
  • Deploy web applications

Syllabus

  • HTML Basics
  • CSS Styling
  • JavaScript Fundamentals
  • React.js
  • Node.js
  • Databases (SQL & NoSQL)
  • DevOps Basics
